申请奖学金（续）

本帖隐藏的内容

有一些学校有一些merit-based奖学金，就是根据你成绩好坏给的钱，但是多数学校的奖学金制度是根据你表格上所说明的需要来给的。奖学金办公室就是要准确计算出你自己能够出多少钱。然后这个数字和学校一年所需要的费用（包括学费，住宿费，生活用品，交通，书费等等）的差价，就是你的“需要”。多数学校会尽量满足这个需求，至少会付清学费和住宿费，有一些特别慷慨的学校可能就会都给。

一个标准的奖学金package包括这么几部分：
Grant: 有的时候就叫做scholarship，但是不是根据天赋和成绩的，而是根据你需要直接给你的一笔钱，相当于礼物不需要还。
Loan：有政府的贷款也有学校的贷款。需要之后慢慢还清，但是一半利息比银行的要低。一般在学生毕业后开始还款。
Job：学校的工作项目。一般学校会让学生替学校每周工作8-12个小时。有的学校叫self-help，你可以选择是把它作为job或者loan。

需要奖学金的国际学生要尤其注意，因为大学只有很少一部分的奖学金才会给国际学生。而且给国际学生奖学金的学校也是有限的。所以竞争非常非常激烈。
对于那些需要很多奖学金的，建议申请的学校种类可以适当多一些，不过当然必须以match为标准，这样可以比较奖学金。
申请奖学金，以下的表格需要填写（不同学校可能有不同的组合）
&#8226;       The CSS (College Scholarship Service) PROFILE
&#8226;       The FAFSA (Free Application for Federal Student Aid)
&#8226;       学校自己的奖学金申请表

CSS FINANCIAL AID PROFILE简介和申请指导

CSS FINANCIAL AID PROFILE简介和申请指导
很多学校都把PROFILE作为申请奖学金的申请表格，包括很多奖学金项目也是。PROFILE上的数据将用来决定一个学生是不是有接受学校自己资金的资格。
学生要填写PROFILE必须先注册。College Board提供三种注册方式：
&#8226;       电话（需要信用卡） 800-778-6888
&#8226;       网上注册（需要信用卡）www.collegeboard.com (点击css/profile)
&#8226;       通过邮件（用信用卡或者汇票）
要是要通过邮件，需要先填一张注册表。这张表格会问一些基本的问题：姓名，地址，要把表格送到的学校名字，等等；父母的收入信息在注册表上不需要填。
一旦你确定了要申请哪些学校，把PROFILE注册表和注册费，以及给每个学校送PROFILE表的钱一起寄给the College Scholarship Service。（一次最多10个学校。）学生必须在各个大学的截止日期前完成PROFILE。有些学校早申请的表格可能9月份就要提交了。所以截止日期早点留意。

提交注册表的两周内，你会收到一份PROFILE申请包裹，里面有表格需要填写资金状况，一些学校要求的额外表格等等。还会有一封信知道哪些地方要填写，什么时候要寄给哪个学校。把包裹里面的所有表格尽早填好，然后提交。多数学校还会要求同时上缴税单，国内没有税单的话解释一下，并且提交一下父母工资收入证明。那个要到父母单位去开。

多数学生会发现网上填写是最快速的方法。所以只要可以，建议使用网络系统。更多关于PROFILE的信息和FAQ在College Board网站上可以查询。

在你提交了PROFILE申请表后，你会在不久后收到the Data Confirmation Report。把这张东西妥善保管。

哪些额外的东西可能要提交？
&#8226;       DIVORCED-SEPARATED PARENT’S STATEMENT，如果父母离异，需要填写这张表格。这张表填写好，复印好，不需要送到CSS，直接送到申请学校。
&#8226;       BUSINESS.FARM SUPPLEMENT。如果父母自己开公司或者是农民，需要填写这张表格，同样，直接送到各个申请学校。

一。招生办的工作

招生办的工作
二十五年前，大约50%的美国高中生进入大学。2001年，这个数字式62%。好学校的竞争非常激烈。多数学校的申请人数一直在增加，录取率一点点下降，这个趋势会一直延续下去。比如说：
&#8226;       Boston University08届增加了2000个申请者，但同时把它们新生的人数减少了500人。
&#8226;       Connecticut College受到4392份竞争515个名额，录取率34%。
&#8226;       哈佛08年第一次突破了20，000大关的申请者，录取了1600多个早申请的申请者。
&#8226;       NYUshenqing者自1991年以来翻了三倍，录取率从65%下降到27%。
&#8226;       斯坦福十年前的录取率是22%，现在是12%。

不同大学收人的方法是不一样的。许多大的州立学校做决定仅仅是按着一条很机械化的标准。成绩，分数，带入一些公式，（这些公式对本州学生和州外学生常常很不一样），然后根据这个公式的结果收人。运动员以及一些其他特殊人群会受到特别照顾。除了成绩这类客观数据之外的很多别的因素就不那么重要。

私立学校就不一样了。他们很注重学生的学术成就，同时也一样注重其他因素，使得招生人员能够更清楚全面地了解一个申请者是什么样的人。推荐信，学生作文和面试，学校报告和课外活动以及领导能力都会被仔细的考虑。

奖学金办公室的工作

奖学金办公室的工作
各个学校奖学金办公室的工作方式也是不一样的。包括他们给每个申请者的钱，package里grant, loan, job的比例也不一样，因为各个学校分给奖学金的资金储备就不一样。奖励性质的奖学金是很有限的。多数是按需求分配的。学校的总奖学金数目取决于校友捐款。
招生办公室的工作就是觉得如何最公平的分发奖学金。首先，现决定所有的学生需要多少，然后决定如何分配各个部分在package里面的占有比例。

奖学金需要是如何被决定的？

父母贡献：首先，所有的收入和财产，包括银行存款，证券，各项投资等等，以确定家里的总资产。然后，计算出要开消：多少孩子要读大学，家中欠债，必要的花费，纳税等。同时也会考虑特殊因素，比如突然有人生病，或者一个孩子需要特殊的教育。一定要确保学校注意到你家里的任何特殊情况。这个差值就会成为在你奖学金通知单以及i-20上显示的family contribution.
还要注意学校对父母离异，分居等等有不一样的计算方法。有的学校会把继父母的收入也算入内，有的不会。这些也是要搞清楚的。

学生贡献：你挣的钱和你的财产也会被考虑在内。典型来说，你会需要贡献你自己的财产和存款的一部分。不管你是不是会在暑假打工，学校多少会让你出一点暑假可能打工的钱。不过国际学生一般学校不会让你出这部分。如果你从学校外的其他机构拿到奖学金，需要及时报告给学校。一般这部分的奖学金，学校不会因此减少你的grant，但是会减少你的loan，所以报告不会有什么不好的地方。

一旦你的家庭贡献确定了，把这个数字从总花费里面减掉，就是你demonstrated need。理想的话，就是这部分need都被满足，不然的话，就是有一个gapping。
