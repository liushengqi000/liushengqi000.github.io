今天MSN居然又出现登录错误“80072efd”，卸载重装亦不能登录，上周也是发生此种错误，无奈重装系统。今天实在心有不甘，于网上搜索到解决办法两种，记录如下：

方法一：

1. 在任务栏的通知区域（时钟附近），单击 MSN Messenger 图标，然后单击“退出”。
2. 通过下列操作“清除安全套接字层 (SSL)”：
a. ie浏览器
b. Internet 选项。
c. 单击“内容”选项卡。
d. 在“证书”下，单击“清除 SSL 状态”。
e.  收到确认信息时，单击“确定”。
3. 通过下列操作预注册和“加密服务”相关的 DLL 文件：
a. 在任务栏上，单击“开始”，然后单击“运行”。
b. 复制下列文本，并将其粘贴到“打开”文本框中：
regsvr32 softpub.dll wintrust.dll initpki.dll
c. 单击“确定”。
d. 收到确认信息时，单击“确定”。根据计算机情况不同，第三次确认信息可能要过几分钟显示。
4. 第三次确认后，重新启动 MSN Messenger 并再试尝试登录。

方法二：

MSN的Option-connection里面的advanced settings中，将socks,http,proxy的内容都清空，方框里的勾也去掉只留下TCP,然后test connection

另：windows自带的messager卸载方法：

开始&gt;运行 键入
RunDll32 advpack.dll,LaunchINFSection %windir%\\INF\\msmsgs.inf,BLC.Remove
XP 系统验证通过

MSN登陆错误，错误代码80072efd

     两个同事登陆msn时都碰到了登陆不成功的事情。错误代码都是80072efd。这个错误代码的含义就是网络连接有问题，msn无法连接到msn服务器；两个同事碰到的问题如下：

      一个是在ie的选项里的连接里代理服务器的选项给选上了。但是并没有设置具体的代理服务器的地址造成的；

        一个是在adsl虚拟拨号的设置了该拨号使用以下代理服务器地址的选项也选上了，但是也是没有设置具体的代理的地址，造成的。感到困惑的是第二种情况里，上网浏览玩网络游戏等都是可以的，单单不能登陆到msn messager。

解决问题的办法如下：

To remove proxy settings in MSN Messenger: go to &quot;Tools&quot; at your MSN Messenger main window then go to &quot;Options&quot; and click the &quot;Connection tab&quot; in the options screen. If there is there is a proxy enabled untick it and try if you can sign in now.

In Internet Explorer:
Tools - Internet options - press &quot;Lan Settings...&quot;
In the screen that appears next make sure &quot;Use a proxy server&quot; is unticked.

Note that the Internet Explorer proxy settings affect more than Internet Explorer itself so you should make sure it is correct even when you user another browser such as Firefox or Opera.
