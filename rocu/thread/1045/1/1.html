3月1日 水仙 (Daffodil)
花语 : 傲慢
花占卜 : 您天生具有高贵的气质，自觉不应埋没自己的天份，对人生很认真，相信必有出人头地的一天。您虽然自命高贵，但很有进取心，以高傲的姿态面对人生，行为有时有点怪异，但肯定是个有为的青年。
花箴言 : 偏见是迂腐，傲慢是精神上的自我高攀。

3月2日 金凤花 (Butter Cup)
花语 : 道德观
花占卜 : 您心地善良正直，不会怀有害人之心，您喜欢追求精神上的解脱，不为物欲所动，所以您是宗教拥护者。在爱情上，您注重心灵交流，属于精神恋爱的类型。
花箴言 : 知恩图报，不要辜负爱人对您的一片心意。

3月3日 紫云英 (Milk Vetch)
花语 : 幸福
花占卜 : 您是一个理智、冷静的人，为人正直，获得朋友的支持和信赖。但您欠缺一种成熟的魅力，您需要透过人际关系来增加您的号召力，在爱情上不宜急进，幸福会一直伴随着您。
花箴言 : 幸福不是必然的，只要您懂得珍惜，幸福唾手可得。

3月4日 木莓 (Raspberry)
花语 : 爱情
花占卜 : 您很有使命感，经常觉得自己责任重大，有点神经质。被您关注的人是个幸运儿，您会尽心尽力地照顾他人，加上您有一颗仁慈、勇敢的心，您的人生将会很充实。
花箴言 : 有使命感的人，人生会是丰盛和有意义的。

3月5日 矢车菊 (Corn Flower)
花语 : 幸福
花占卜 : 宽厚仁慈的您，对人总是热情大方、心胸广阔，不会计较别人的过错，以真诚待人，因此深受大家爱戴。您亦享受和谐的生活，知足常乐，沐浴在幸福之中。
花箴言 : 懂得知足，是人生最大的幸福。

3月6日 雏菊 (Daisy)
花语 : 开朗
花占卜 : 不知不高地厚的您，带着天真、无邪的本性，游戏人间。基于您开朗的性格，容易相信别人，也容易上当受骗。尤其在感情上，您应该小心一些，否则感情易放难收，到头来伤害了自己。
花箴言 : 虽然天真烂漫是您的本性，但也会学会保护自己。

3月7日 小豆蔻 (Cardamom)
花语 : 激情
花占卜 : 您是一个率直的人，不会掩饰自己的感情，有时更会激动得在梦中流泪。您一旦动了真情，往往有轻率的举动，不懂得控制自己，反而令对方避开您，您应注意与人交往是需要点耐性的。
花箴言 : 激情握于手?容易枯死，藏于心内则历久常新。

3月8日 栗子花 (Castanea)
花语 : 真心
花占卜 : 您对于自己的条件充满自信，处事果敢决断，才能卓越，敢作敢为，但相反您亦是一个孤独的人，常因饶舌而引起误会。您应该放开胸怀，接纳别人的意见，那么您的真心真意亦会很快被人接纳。
花箴言 : 即使您真心待人，也要敞开心扉，容纳他人。

3月9日 落叶松 (Larch)
花语 : 大无畏精神
花占卜 : 您很有爱心及勇气，属于耐力强及勇于挑战困难的人，时常有一些出人意表的举动。虽然您胜算在握，但总是嬉皮笑脸，给人玩世不恭的感觉，其实您内心十分浪漫，只是旁人不察觉而已。
花箴言 : 您的大胆令人震惊，您的勇敢更令人刮目相看。

3月10日 榆树 (Elm)
花语 : 高贵
花占卜 : 您心胸广阔，不拘小节，是个性情中人。您对爱情很敏感，喜欢率直认真的人。虽然爱您的人很多，却不易找到意中人，您需忍耐多一阵子，等待爱神帮您撮合吧。
花箴言 : 缘份不可强求，只可以等待。


3月11日 菊苣 (Chicory)
花语 : 慎重
花占卜 : 您是属于小心谨慎的类型、与人交往有情有义，给人稳重的感觉。您谈恋爱亦是小心翼翼，很少会有一见钟情的例子，当一切条件成熟时才会表露爱意，绝不轻易移情别恋。
花箴言 : 谈恋爱切勿像昙花一现般，一瞬即逝。

3月12日 柳树 (Weeping Willow)
花语 : 愁伤
花占卜 : 您个性温柔，富浪漫气息，有点不切实际，喜欢追求柏拉图式的爱情。由于您多愁善感，易被爱情所困，经常显得心事重重，您应该训练自己正确面对问题，不要为了小事而自怨自艾。
花箴言 : 人生不如意事十常八九，伤春悲秋解决不了问题。

3月13日 萱草 (Day Lily)
花语 : 难忘的爱
花占卜 : 您是天生的老好人，总是顺从别人的说话，失去自己的主见。您个性容易被人利用，吃亏上当在所难免，所以您应建立自信心，有了自信，爱情会顺心顺意得多。
花箴言 : 勇敢地拒绝别人无理的要求，是件可喜的事情。

3月14日 杏仁花 (Almond)
花语 : 希望
花占卜 : 您是个感性的人，虽然不善辞令，却是个诚实的君子。您对爱情很专一，并相信情人许下的诺言，只要对方不变心，您是可以从一而终的人。
花箴言 : 您心中有很多盼望，天长地久的爱情只是其中一个。

3月15日 铁杉 (Hemlock)
花语 : 无悔
花占卜 : 勇敢、果断的您，处事不会拖拖拉拉，想做就去做，给人一种爽快、率性的感觉。您容易为爱疯狂，愿意为爱人牺牲，义无反顾地奉献一切，但若要终止时，亦不会犹豫。
花箴言 : 爱不可过量、而是要洽到好处。

3月16日 薄荷 (Mint)
花语 : 美德
花占卜 : 您有良好的品德，谦虚有礼，既懂得迁就别人，亦有自己的性格。您很注重别人对您的评价，旁人的意见是您积极改善自己的根源。感情上您处理得宜，甘苦与共是您对爱情的心得。
花箴言 : 当您遇到挫折时，放两片薄荷叶在眼前，可以看清问题。


3月17日 豆荚花 (Bean)
花语 : 必然的幸福
花占卜 : 您是开放的乐天派，追随流行节拍，是个新潮感的人，跟您一起的人都会被您的乐观开朗感染。虽然如此，您对爱情很专一、认真，绝不滥交，真爱到来时，您必定会得到幸福。
花箴言 : 只要您付出真感情，幸福是必然的。

3月18日 芦笋 (Asparagus)
花语 : 物尽其用
花占卜 : 您是一个率直的人，什么事情都会表露出来，因此您容易受到伤害，但磨练的机会亦会增多。在面对失败的时候，您懂得自我安慰和勉励，从失败中学习，所以您的人生经验很丰富。
花箴言 : 若不经过磨练，又怎会人尽其才，物尽其用呢？

3月19日 栀子花 (Cape Jasmine)
花语 : 喜悦
花占卜 : 您有感恩图报之心，以真诚待人，只要别人对您有少许和善，您便报以心灵致谢。这是因为您有一颗赤子之心，不懂人心险恶，而您的真诚使您常怀欢愉，宽恕他人也使您充满喜悦。
花箴言 : 有时候要把自己的喜恶表现出来。

3月20日 紫色郁金香 (Tulip)
花语 : 永恒的爱
花占卜 : 您拥有美丽的气质，喜欢追求爱情，积极维护两个人的世界。您为了得到永恒的爱情，会不断地寻心目中的理想情人，一次又一次地恋爱，只要您提高自己的智能，真爱就在不远处。
花箴言 : 其实，永恒的爱就在您对某个心仪的对像示爱时产生。

3月21日 樱兰 (Honey Plant)
花语 : 人生起点
花占卜 : 您是个含蓄、内向的人，不敢将自己的感情流露出来，常常言不由衷，面对爱人时，您总是不敢吐露自己的真心意，属于爱在心?口难开的类型。不妨将心事说出来，勇敢地示爱。
花箴言 : 恋爱是令人成熟的人生起点。

3月22日 锦葵 (Mallow)
花语 : 恩惠
花占卜 : 您外表坚强，个性温柔，是家庭的中坚分子，负起照顾家庭的责任。您喜欢细水长流的爱情，与爱人慢慢培养感情，这份稳定的关系，使您和您的另一半更有安全感。
花箴言 : 细水长流的感情正是永恒的爱情。

3月23日 剑兰 (Gladiolus)
花语 : 热恋
花占卜 : 您天性好动，爱冒险犯难，愈是困难或不可为的事情您愈有兴趣去参与，所以您很容易卷入情感纠纷之中，或者爱上您不该爱的人。要解决问题，您必须以开朗灿烂的笑容去阻挡不必要的恋情。
花箴言 : 如果您恋上这种花，则暗示您有外遇或三角恋情。

3月24日 加州罂粟花 (California Poppy)
花语 : 希望
花占卜 : 您拥有聪明才智，但不懂得发挥，欠缺组织能力，所以遇到困难时显得忧虑和逃避，容易放弃，导致错失良机。您很希望改善这种情况，只要您勇敢一点，事情是会有转机的。
花箴言 : 当您真心面对自己的挫败时，您会发觉问题是有转机和希望的。

3月25日 藤蔓植物 (Climbing Plant)
花语 : 美丽
花占卜 : 您非常善解人意，但在感情上收获并不丰富。虽然缘份是可遇不可求，不过您需要积极争取，机会有时是可以制造出来的。您想自己拥有迷人的吸引力，必须努力地提升自己的才华。
花箴言 : 美丽是包含外在美，同时也有内在美。

3月26日 白色樱草花 (White Primrose)
花语 : 初恋
花占卜 : 您认真诚实的性格以及良好的道德观，得到朋友的信赖，加上您天赋的才能，有着很好的人生路。不过，您这人很念旧，尤其是对初恋情人，对于过去的风花雪月仍然挥之不去。
花箴言 : 逝去的爱情如枯萎的花儿般不可再生，未来的爱情充满希望。

3月27日 荷包花 (Calceolaria)
花语 : 援助
花占卜 : 您是个专情专一的人，对爱情的态度审慎，不会轻易谈恋爱，一旦对上了，您会至死不渝地爱下去。如果您发觉情人对您不专，您是无法原谅他的，所以您需要结交多些朋友。
花箴言 : 婚姻的价值是要令人变得更成熟。


3月28日 圆珠花 (Robinia Hispida)
花语 : 高尚
花占卜 : 您只知道对人好，为人切想，生怕得罪了别人。当您面对感情问题时，您总是觉得自己很渺少，对方很伟大，不知不觉中流露您的自卑感，要明白爱是对等的，您有高尚的人格，为什么要贬低自己？
花箴言 : 不要夸大别人的好处，轻视自己的长处。

3月29日 牛蒡 (Burdock)
花语 : 烦扰
花占卜 : 您是个不辞劳苦、埋头苦干的人，遇到困难不会轻易落泪，咬紧牙关坚持到底是您做人的宗旨。不过一旦涉及金钱和暴力，您就会沉不住气，所以您应冷静明智一点，别做一只蛮牛。
花箴言 : 无视困难、勇往直前是好事，却不要做横冲直撞的蛮牛。

3月30日 金雀花 (Broom)
花语 : 整洁
花占卜 : 您天性纯良，待人和蔼可亲，比较容易为人接受。您处事很有条理，办事干脆利落，每当有一点小成绩时，您都会为自己感动，相当惹人疼爱。偶尔表现出来的进取精神，对您更有好处。
花箴言 : 每个人都应该为自己的成就喜悦。

3月31日 黑种籽草 (Black Caraway)
花语 : 梦幻爱情
花占卜 : 您依然留恋过去的恋情，跳不自己制造的感情框框。您虽然也希望得到新恋情，但总是一逃三避，爱情只会昙花一现。与其等待被爱，不如主动地去爱别人，积极开始您的人生。
花箴言 : 傲慢的人是无法恋爱的。
