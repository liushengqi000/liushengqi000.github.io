In a community of 416 people, each person owns a dog or a cat or both. If there are 316 dog owners and 280 cat owners, how many of the dog owners own no cat?

36
100
136
180
316

[glow=255,red,2]Hint[/glow]
Even though finding 180 is a key step in arriving at the solution, 180 is not the final solution. The temptation may be to answer incorrectly with choice D. Remember to check that you have answered the question that was asked.

[glow=255,red,2]Answer[/glow]

To solve this problem you must realize that some of the 316 dog owners also own cats and some of the 280 cat owners also own dogs. This has to be true because 316 + 280 is more than 416! The question asks how many of the dog owners do not own cats. To solve the problem, first find the number who own both a dog and a cat. To do this let b equal the number who own both a dog and a cat. Then 416 = 316 + 280 – b. (Note: Adding 316 and 280 counts b twice, so b must be subtracted from the sum to get 416.) Next, the equation 416 = 596 – b can be solved for b, yielding b = 180. Finally, the number of dog owners who own no cat is 316 – 180 = 136.

Difficulty: Hard

If the graph of the function &#402; in the xy-plane contains the points (0, –9), (1, –4), and (3, 0), which of the following CANNOT be true?

The graph of &#402; has a maximum value.
y  0 for all points (x, y) on the graph of &#402;.
The graph of &#402; is symmetric with respect to a line.
The graph of &#402; is a line.
The graph of &#402; is a parabola.

[glow=255,red,2]Hint[/glow]

If three points in the xy-plane lie on a line, the slope of the segment connecting any pair of the points must be the same as the slope of the segment connecting any other pair of the points.

[glow=255,red,2]Answer[/glow]
D is CORRECT! Here&#39;s why:
If the graph of the function &#402;, which contains the points (0, –9), (1, –4), and (3, 0), were a line, then the slope of the segment connecting the points (0, –9) and (1, –4) would be the same as the slope of the segment connecting the points (1, –4) and (3, 0). However, the slope of the segment connecting the points (0, –9) and (1, –4) is 5, and the slope of the segment connecting the points (1, –4) and (3, 0) is 2. Therefore, the graph of &#402; cannot be a line.

The statements in the other four options could be true. For example, if the equation of &#402; were y = –(x – 3)2, then the graph of &#402; would contain the points (0, –9), (1, –4), and (3, 0). The graph of &#402; would be a parabola symmetric with respect to the line with equation x = 3. The maximum value of &#402; would occur at the point (3, 0), and y  0 would be true for all points (x, y) on the graph of &#402;.

If (x + y)2 = x2 + y2, which of the following statements must also be true?

I. x = 0
II. (x – y)2 = x2 + y2
III. xy = 0

None
I only
II only
III only
II and III

[glow=255,red,2]Hint[/glow]
This question uses what is referred to as the Roman numeral answer format. This format is used in both math and Reading Passage questions. The way to approach these is to work on each Roman numeral as a separate true/false question. Once you have decided (and marked) each Roman numeral as true or false, it&#39;s easy to find the correct answer.

In this problem, it is helpful to rewrite (x + y)2 as x2 + 2xy + y2.

[glow=255,red,2]Answer[/glow]
E is CORRECT! Here&#39;s why:

The quantity (x + y)2 can be expressed as x2 + 2xy + y2. If (x + y)2 = x2 + y2, then 2xy= 0 and xy= 0. Since xy=0, then either x = 0 or y= 0 or both. Therefore, statement III must be true, but statement I, x = 0, is not always true. For statement II, (x – y)2 = x 2 – 2xy + y2 and, since xy = 0, (x – y)2 = x2 + y2. Therefore, both statements II and III must be true.
