AUGUST


8月1日 红色罂粟花 (Poppy)
花语 : 安慰
花占卜 : 您有梦想家的倾向，满脑子遐思，加上开朗的性格，是个交际高手。您有点虚荣心，耐性不足，有时为了表现自己，有不顾一切的冲动。您应该接纳朋友的忠告，收敛一下容易厌倦的本性。
花箴言 : 赴汤蹈火式的恋爱通常只有三分钟热度。

8月2日 矢车菊 (Corn Flower)
花语 : 幸福
花占卜 : 您有很强的好奇心和好胜心，无论学业或者事业都不会吃苦，而且您精力过剩，喜欢服务别人，是个受欢迎的乐天派。但您热心助人的性格可能冷落了您的情人，好胜心亦会带来不平衡，您要好好把握属于您的幸福啊。
花箴言 : 矢车菊总是用来扰乱别人的视线。

8月3日 金盏花 (Flower of an Hour)
花语 : 少女的美态
花占卜 : 您是个思想单纯的人，并且有一颗不知人间险恶的心，您仍像以往一样，带着浪漫的眼光看世情，所以容易受伤害。可能正是因为您的纯真，身旁的人会特别关心您，给您勇气和希望。
花箴言 : 男人希望自己是她的初恋，女人则希望自己是他最后一个女人。

8月4日 玉蜀花 (Corn)
花语 : 财富
花占卜 : 您天生拥有王者的风范，而且待人亲切，是众人崇拜的偶像。对您而言，世上没有不可能的事情，爱情上更占尽优势，您是异性心目中的情人，但您钟爱的只有一，记住闪闪缩缩的爱情会破坏您的崇高形象。
花箴言 : 光芒四射的人，有时会被虚荣弄得身不由己。

8月5日 石南花 (Heather Ling)
花语 : 孤独
花占卜 : 您有丰富的才能，明确的人生目标，还有很好的交际网，外表看来是个活跃份子，私底下却是个寂寞的人。您有恩必报，对爱情态度认真，有时对自己要求过于严格，自己孤立自己。留点空间吧，不要自虐了。
花箴言 : 趁自己还未自虐死掉之前赶快恋爱吧。

8月6日 喇叭花 (Trumpet Flower)
花语 : 名誉
花占卜 : 您有天真无邪的性情，却喜欢模仿大人的精明干练，使您看来非常有趣。您很有异性缘，喜欢追求大人们成熟的爱情，但入世未深的您又如何体会成年人的世界呢？还是放自然一点吧，您的心会更自在。
花箴言 : 小鬼扮大人，装腔作势。

8月7日 石榴花 (Pomegranate)
花语 : 成熟美
花占卜 : 您浑身洋溢着艺术气息，有着艺术家的执着，一旦决定了的事情必会实行，不会轻易妥协。您高傲的性格，容易丧失朋友，虽然曲高和寡，但更懂得爱的真意，您的美需要懂得欣赏您的人才发掘得到。
花箴言 : 一旦堕入情网，不是欢欣就是痛苦。

8月8日 杜鹃花 (Azalea)
花语 : 爱的喜悦
花占卜 : 纯真无邪的心是您独有的魅力，掳获了不少人的心，同时亦容易被人利用。有时您会无缘无故地感到寂寞哀伤，希望得到别人的关注，但是您不懂得分辨人心好坏，反而惹来更大的伤害。
花箴言 : 当见到满山杜鹃盛开，就是爱神降临的时候。

8月9日 木犀花 (Cistus)
花语 : 人缘
花占卜 : 您与生俱来的魅力，广受大众欢迎，您习惯被人爱护，惯了别人对您献殷勤，反而忘记了对人应有的关心。您必须注意自己的言行，别沉溺于被爱的幸福中，不然当您遇到意中人时，对方可能一点也不在乎您呢。
花箴言 : 记住、被爱是一种奢侈的幸福，会使人麻木。
8月10日 苔藓 (Moss)
花语 : 母爱
花占卜 : 您是个成熟稳重的人，知性与理性的结合，使您懂得随机应变，体谅和宽容他人，您的爱心如母爱般宽宏大量，您的家人和朋友都会深深感受到被爱的幸福。
花箴言 : 不求回报的爱、正是苔藓给予大地最伟大的爱。

8月11日 红色天竺葵 (Geranium Zonal)
花语 : 幸福
花占卜 : 您是一个很有同情心的人，见义勇为，只要有人向您提出请求，您一定会想尽办法去帮助别人。您同样带着忧郁寂寞的神情，故此安慰别人也就等于帮助了自己，您应该乐观一点，积极找出自己的快乐泉源。
花箴言 : 超越自己，也就超越了平凡。

8月12日 夹竹桃 (Oleander)
花语 : 重情
花占卜 : 您有灵巧敏捷的心思，常有新奇的创意，很适合这个商业社会。您对人有很大的戒心，宁取敬而远之的态度，不会轻易打开心扉容纳他人，感情上您重视友情多过爱情，这也是您对人小心谨慎的缘故影响所致。
花箴言 : 不重恩情的人不配做朋友。

8月13日 麒麟草 (Goldenrod)
花语 : 戒备
花占卜 : 您缺乏主见，容易相信别人的谎言，纷争无日无之，令您不胜烦扰。其实您自己也有满腹心事，遇到问题总是太过认真去接受，反而使问题复杂化，只要您别再着眼于那些琐碎事情，您会轻松愉快得多。
花箴言 : 做个精明的人，不要被虚伪的世界蒙骗自己。

8月14日 石蚕 (Wall Germander)
花语 : 敬爱
花占卜 : 您是个灵巧多变的人，适合做生意。初相识的时候给人亲切和善的印象，但其实您非常功利，只不过在您的包装掩饰下不易被人察觉。虽然您的生活充实，但您觉得人生乏味；爱情上虽处处逢源，但难获真心。
花箴言 : 太过计较金钱的人只有铜臭味。

8月15日 向日葵 (Sun Flower)
花语 : 光辉
花占卜 : 您个性坚强，有自己的性格，常常被人误会是生性傲慢，其实您自知才能仍未发挥出来，有志难伸，郁闷在心?。如果您改一改自己的作风，平易近人一些，您的运气会好转一点。
花箴言 : 想得到爱时，先要付出爱。

8月16日 罗梦子树 (Tamarind)
花语 : 奢华
花占卜 : 您天性爱美、好美食，衣着打扮华贵，给人奢华的感觉。然而您本身并不奢侈，只想多方面结织不同种类的人，要有良好的人际关系，多少要有点虚情假意，但别把自己也蒙骗过去。
花箴言 : 想爱却不能，那么您就要检讨一下自己了。

8月17日 百合树 (Tulip Tree)
花语 : 田园幸福
花占卜 : 您有爽朗的性格，从不矫饰造作，崇尚朴实自然的生活。但感情路上并不顺利，可能是您过于随意的打扮，未能配合对方需要，您要考虑一下双方的习惯以及接纳程度达到什么阶段，这样才会顺利的。
花箴言 : 懂得尊重自然界的生存者，是人类的一大进步。

8月18日 独葵 (Hollyhock)
花语 : 热恋
花占卜 : 您有自己的生活规律，但有时过于神经质，惊慌失惜反而乱了方寸。当您遇到困难不知如何是好的时候，您应该找您心爱的人商量，对方走的路可能就是您走向幸福的道路。
花箴言 : 别要贪刺激了，选择正确的人生道路是相当要紧的。

8月19日 蜡烛花 (Rose Campion)
花语 : 诚实
花占卜 : 您将注意力集中在事业的发展上，工作的成果为您带来很大的满足感。您有很强的求知欲、理性加上努力，上司很满意您的工作表现。至于爱情，您的伴侣会被您的热认真深深吸引。
花箴言 : 时间、地域、空间是恋爱是否成功的元素。

8月20日 香雪兰 (Freesia)
花语 : 纯洁
花占卜 : 您天真甜美的脸容就如画中的纯情少女一样，不吃人间烟火，同时亦不知人心险恶。听到甜言蜜语，您显得无力招架，沉醉于如诗如画般的恋爱，这是很危险的，您应赶快学习保护自己，提高惊觉。
花箴言 : 您要知道爱的背后隐藏着快乐与痛苦、悲伤与后悔


8月21日 龙芽草 (Agrimony)
花语 : 感谢
花占卜 : 您有自己的道德标准，对人严格，对自己亦同样严格，同时，看人相当准确，可以找到理想中的对象。不过您占有欲很强，对爱情很专注，对方被您的掌控迫得有点喘不过气，您应该学习将爱情和温柔表现出来。
花箴言 : 爱是需要培养的，不可预先设定各样的标准。

8月22日 绣线菊 (Spirea)
花语 : 努力
花占卜 : 您对自己的实力充满信心，处事果断勇敢，干脆利落，绝不拖泥带水。您的态度是值得欣赏的，但太过独断会被人误会是个独裁的人，有时也要考虑别人的需要，毕境这个世界不是您一个人独有的。
花箴言 : 婚姻是需要互让的，您有这个信心吗？

8月23日 西洋菩提树 (Lime Tree)
花语 : 夫妇之爱
花占卜 : 活泼开朗的您，在遇上伤心失意的时候，意志力显得非常薄弱。您需要找个意志力比您坚强的人帮您一把，同时您自己也要学得坚强、独立一些，不要时时想着倚赖别人帮您解决问题。
花箴言 : 盲目的倚靠别人，走千里路也等于原地踏步。

8月24日 金盏花 (Calendula)
花语 : 别离
花占卜 : 您的生活富裕丰足，在安稳顺逐中成长，可说不懂愁滋味。您喜欢甜蜜温馨的爱情，一旦遇分离，悲伤的情绪难以控制。其实您应该明白，有分离才有结合，从分离那一刻萌生的希望，才是最值得您去珍惜。
花箴言 : 只要两心相连，无惧地域阻隔。

8月25日 火鹤花 (Flamingo Flower)
花语 : 苦恋
花占卜 : 您热情洋溢，愿意为爱情奉献，但仍然停留在幼稚的幻想之中。您快人快语的性格容易被人误会，要学习成熟一点，自然地表现自己就够。
花箴言 : 事业上要保持沉默是金，而爱情上并非沉默是金。

8月26日 小金梅草 (American Star Grass)
花语 : 寻求光明
花占卜 : 您天生多疑，对人戒心很重，人际关系亦不如意。造成这种局面只因您想得太多，何不放开点，结织多些朋友，到时您的观感可能会改变。恋爱不一定是伤心的，也有很多动人的甜蜜故事。
花箴言 : 爱有酸甜苦辣，但爱过总比从没爱过好。

8月27日 弹簧花 (Osmunda Fern)
花语 : 梦想
花占卜 : 您是个动静皆宜的人，静的时候可以一个人默默地沉思，动的时候可以天南地北谈过不休，为大家带来欢笑和希望。无论是独处还是与人共处，您对自己始终信心十足，但您要拋弃那种学术家的口吻。
花箴言 : 青春无价，切勿虚渡。

8月28日 蕈毒花 (Eryngium)
花语 : 秘密恋情
花占卜 : 您爱面子，却不擅长表达自己，明明害怕寂寞，但在您口中会说成是喜欢孤独，常常言不由衷，有苦自己知。您情深款款，明明喜欢别人，却装作若无其事，这样只会令您自己痛苦。
花箴言 : 其实您很清楚，结婚就是您的事业。

8月29日 烟草花 (Flowering Tobacco)
花语 : 全赖有您
花占卜 : 您总是众人眼中的焦点，有您在的场合，一定充满欢乐的气氛。但是，您外表看来虽然像个活泼的娃娃，内心其实是内向和孤独的。不妨打开心窗，将您的心底话告知您的情人，对方可以带给您真正的快乐。
花箴言 : 放弃虚假的一面，会发现新的自己。

8月30日 石蚕 (Wall Germander)
花语 : 淡泊
花占卜 : 天生的第六感赋予您拥有超凡的感应力，凭着这股力量，您的判断会比别人快和准确得多。但是您缺乏恒心和耐性，白白错失了很多时机，您善于帮人扯红线，自己仍处于独身。
花箴言 : 先天的才能也要配合后天的努力，才能事半功倍。

8月31日 苜蓿 (Clover)
花语 : 誓言
花占卜 : 您具备领袖的风采，处事英明果断，对人有求必应，深受众人的爱戴，亦受上司的赏识。您身边有不少的追求者，但您似乎仍在寻寻觅觅，只要将眼光放远些，自有幸福等着您。
花箴言 : 恋爱、恋爱，还是要恋爱。
