DECEMBER


12月1日  艾菊 (Tansy)
花语 : 和平
花占卜 : 您心胸广阔，不拘小节，对朋友够义气，但就是不知为何没有恋爱运。原因是您对爱情的敏感度极低，因而失去了不少恋爱的机会，只要您细心留意一下身边的人，不论财运、蜜运，都离您很近。
花箴言 : 假若不帮植物浇水，它又怎会开花结果呢。

12月2日 苔藓 (Moss)
花语 : 母爱
花占卜 : 充满爱心的您，对人总是千依百顺，同时亦建立了一个广阔的社交圈子。因为身边的人太多，占去了您大部份的私人时间，对您造成很大的困扰。所以，您和朋友应保持距离，让自己有时间轻松一下。
花箴言 : 朋友分几种，您要识别哪些人只是泛泛之交。

12月3日 熏衣草 (Lavender)
花语 : 期待
花占卜 : 您本来是讨人喜爱的，但是比较骄纵任性，不大理会别人的感受，而且要求甚多，恋情最终都是不欢而散。您应学习多付出些耐性，体谅别人的难处，控制一下自己的情绪，这样会较易被人接受。
花箴言 : 玫瑰虽美但带刺，难免给人一种钟看不钟用的印象。

12月4日 酸叶 (Rumex)
花语 : 情爱
花占卜 : 您有开朗的性格，是个不拘小节的乐天派，和朋友一起，总能带起欢乐的气氛。您理智的思维及丰富的知识，有说不完的话题，但遇事不够果断，有拖拖拉拉的无法洒脱，到头来悔之已晚。
花箴言 : 钱财身外物，只有爱情是您唯一的期待。

12月5日 安布罗希亚 (Ambrosia)
花语 : 幸福
花占卜 : 您是个冷静及爱思考的人，凡事喜欢追根究柢，直至找出真正的原因才会作罢。同时，您也是个公平的正义者，您心?面一直算计着自己的得失，这样显得您太过小孩子气了，爱情有时也需要大方一点。
花箴言 : 不在身边的人，思念会更加强烈。

12月6日 虎耳草 (Saxifrage)
花语 : 切实的爱情
花占卜 : 您有过人才华，求知欲强，冷峻的外表下，隐藏着如明星般的吸引力。您有冷静的思想，因此不会沉迷爱情；您希望拥有知心的恋人，可惜您高傲的性格令您的情路并不顺利。只要稍加改正，必会遇到理想情人。
花箴言 : 不要再伪装坚强了，快找个合心意的人吧。

12月7日 蕨 (Fern)
花语 : 信赖
花占卜 : 您是个有智能、说话风趣幽默的可人儿，加上您独特的思想及对人认真的态度，赢得了朋友的信任。您的爱情比较顺利，有不少追求者，但您必须学习倾听别人的说话，别要只顾表现自己。
花箴言 : 既然喜欢别人谈论自己，也要乐意接受别人的批评。

12月8日 芦苇 (Reed)
花语 : 深情
花占卜 : 您天生有丰富的才华，而且为人颇自负，禁不起别人的怂恿，勇于向人挑战。您喜欢听到赞美的声音，失去了原有的自我本色，加上容易相信别人，误交损友。积极培养一下观察能力，您有获得幸福的充分条件。
花箴言 : 注重内在美吧，虽然是平常人看不见的地方。

12月9日 菊 (Chrysanthemum)
花语 : 高洁
花占卜 : 您如天使般受人敬爱，加上率真开朗的性格，令不少人成为您的裙下之臣。您的爱情是纯白无垢的，留意一下身边的朋友，当中的一个正是您的真命天子，他热烈的爱情必能感动您。
花箴言 : 得到完美的爱情，是人生最大的喜悦。

12月10日 红色山茶花 (Camellia)
花语 : 理性
花占卜 : 您是个急性子的人，凡事只想知道结果，对于中间的过程反而不太注视，因此您比别人总是要付出双倍的代价才能得到同等的收获。您希望拥有热烈的爱情，早已不留退路地下定决心，直至找到真爱为止。
花箴言 : 千万不要意气用事，投石问路是一种自保的的方法。

12月11日 松叶菊 (Fig Marigold)
花语 : 爱国心
花占卜 : 您是个和平主义者，充满博爱精神，对家人、朋友都有一颗怀柔的人，并热心参与公益活动。在爱情上您比较被动，缺乏求爱的勇气，如果您可以拿出勇气来，您的生活会更加美满、充实。
花箴言 : 「和平共处」、「守望相助」是您的基本准则。

12月12日 棉 (Cotton Plant)
花语 : 优秀
花占卜 : 您一直都有出色的表现，无论读书时代抑或工作，都是团体?的火车头，您的领导才能是众人公认的。您关心别人的本性，令您平添了几分温柔。您对爱情要求很高，一定会找一个懂得欣赏您的人为止。
花箴言 : 想爱就要去爱嘛。

12月13日 紫红色菊花 (Chrysanthemum)
花语 : 爱情
花占卜 : 您有朴实内向的性情，而且充满人情味，喜欢照顾别人的感受，是个受欢迎的人.。由于您有害羞的性情，感情上处于波动，只要您相信自己的直觉，积极一点，一定得到情人的贴心宠爱。
花箴言 : 菊花最是多姿多采，接近它多一些，人也活泼一点。

12月14日 松 (Pine)
花语 : 勇敢
花占卜 : 今天生日的您，拥有强健的体魄，是个优雅而威严的人。您讨厌懦弱的行为，喜欢刚强有劲的物体。感情上您需要温柔体贴一些，撒娇有时可以增加您的魅力，对另一半信任，才能走上幸福之路。
花箴言 : 我的心就如松柏般光明磊落。

12月15日 紫丁香 (Winter Daphne)
花语 : 不灭
花占卜 : 您沉默寡言，不喜浮夸，是个踏实、自爱的人。但您也有感性的一面，珍惜永恒的爱情，将意中人的心意埋于心底。可是，对被爱者而言却感受不到这种幸福，您需要坦白一点，那么对方才能接收您的信息。
花箴言 : 将情人的方香藏于心中，是有点可怜的单恋。

12月16日 赤杨 (Alder)
花语 : 庄严
花占卜 : 您有美丽的容貌，思想单纯，容易受人诱惑。您对爱情有很多幻想，只要异性对您笑一笑，就会心花怒放，不其然地堕入恋爱的云雾中。这点您应该反省一下，爱情不是闹着玩的，否则遍体鳞伤的是您自己。
花箴言 : 从恋爱到结婚，需要经过一段漫长的路。


12月17日 樱兰 (Honey Plant)
花语 : 同感
花占卜 : 您是个追求美感的人，加上纤细感性的特质，做任何事情都能够细心地逐一完成。您明白两人之间的爱情，必须要有共通的道德标准，真诚以对，肯为爱情牺牲，才会有美好的前景。
花箴言 : 有花也有果实的人生，才是丰盛的人生。

12月18日 鼠尾草 (Sage)
花语 : 家庭观念
花占卜 : 您重视贞节，为人光明磊落，是个正人君子型的人。您热爱家庭，具有知性的一面，而且极具创造力，被您爱护的人享有十足的安全感。您为了达到自己的理想，不惜消耗时间和体力，属于大器晚成。
花箴言 : 正直的人总是要多绕几个圈子的。

12月19日 雪片莲 (Snowflake)
花语 : 美丽
花占卜 : 谦虚、诚实的您，赢得了朋友的信赖，而且您有乐于奉献的精神，为他人忙?忙外，从不皱一下眉头。被您爱护的人可真有福了，对方除了可以享受您那份纯洁的爱情之外，还可以拥有您美丽的外表。
花箴言 : 充实的生活令人生不再平淡。

12月20日 菠萝 (Pineapple)
花语 : 完美
花占卜 : 您是一个近乎完美的人，有丰富学识，仪表出众，而且善于交际，您的朋友应该不少。虽然如此，您缺乏知心好友，原因是您一直以高傲的心对待他人。至于爱情，也是因为您的理想过高，所以至今仍未找到真命天子。
花箴言 : 完美的人寻觅对象也是不容易。


12月21日 薄荷 (Mint)
花语 : 美德
花占卜 : 您是个平和的人，不喜欢争强好胜，凡事总是顺应别人的意思，缺乏自己的主见。幸亏，您温和的性情给人良善的印象，而且很有异性缘。感情上还算顺利，不过您必须增强您的主见及勇气。
花箴言 : 人的意识主宰了自己一生的幸福。



12月22日 百日草 (Zinnia)
花语 : 幸福
花占卜 : 您是个娇生惯养的可人儿，生命中有享不完的幸福。惹人注视的您害怕寂寞，因此您很重视友情，最担心友谊日趋淡薄。另外，您有很多机会谈恋爱，应该是不愁寂寞的人。
花箴言 : 爱花的人，自有惜花之人。

12月23日 筱悬木 (Phtanus)
花语 : 天才
花占卜 : 您与生俱来的本领，加上后天的努力，是个十足十的天才，人们对您的才能非常景仰，而您对自己的人生亦是充满信心和希望。与您相爱的人会在适当的时间出现，您们拥有的共同智能，会谱出一段美丽的恋曲。
花箴言 : 天才也需要后天的努力，才能发挥天份。

12月24日 檞寄生 (Mistletoe)
花语 : 耐性
花占卜 : 您有很好的意志力，不为任何诱惑所动，有坚定人生目标。您有克服困难的魄力，耐心地等待时机，不会轻言放弃。要注意，您最大的敌人就是自满，高傲的人容易离群，孤掌难鸣是失败的导火线。
花箴言 : 自大的人是不会拥有玫瑰色的瑰丽人生。

12月25日 冬青 (Holly)
花语 : 先见之明
花占卜 : 您是个头脑清晰、处事有计划、有分寸的人，不会受旁人影响，事情发展都在您的掌握之中。您对自己充满自信，借着洞悉先机的能力，您的成功总是快人一步。您的伴侣会是一个充满挑战性的人。
花箴言 : 疑神疑鬼的人只会弄得心绪不宁。

12月26日 圣诞蔷薇 (Christmas Rose)
花语 : 追忆
花占卜 : 您思想单纯，喜欢幻想，渴望有一段如灰姑娘般的遭遇，邂逅心中的白马王子。您天生直觉敏锐，能够看穿对方的心意，而且善于说话。可是爱情总是离您远远的，原因是您太过梦幻，给人不切实际的印象。
花箴言 : 沉迷幻想，在成熟与幼稚之间摸索前行。

12月27日 梅 (Plum Blossom)
花语 : 澄澈的心
花占卜 : 您天生有独特的魅力，如艺术家的气质，高贵及高傲，有点固执及自我。您有已定的社交圈子，朋友差不多都是同一类型，所以朋友不多，您的对象也会是和您同一类型的人。
花箴言 : 梦见梅花，表示会有幸运的事情发生。

12月28日 石榴花 (Pomegranate)
花语 : 成熟的美丽
花占卜 : 您朴实无华的生活方式，给人一种老气，过时的感觉，但不要紧，这只是平庸的人一般见识。懂得追求人生真、善、美的人，才是真正懂得生活的人，别被闲言闲语影响您的人生目标。
花箴言 : 传说中吃石榴可以去除心中的嫉妒与憎恨。

12月29日 灯笼草 (Winter Cherry)
花语 : 自然美
花占卜 : 您是个崇尚自然的人，不喜矫楺造作，更不会追求名牌，勤俭、朴素是您的人生标准。在爱情上，您偏向选择自然、豪情的人，而且表现得小心翼翼，谨言慎行，发展应该会很顺利。
花箴言 : 幸福由对他人的宽大而生。

12月30日 蜡梅 (Carolina Allspice)
花语 : 慈爱
花占卜 : 您善于关心他人，而且有为他人奉献、牺牲精神，是个典型服务社会的人士。您有颗浪漫的心，又过于相信别人，容易被人利用。只要您在看人的本领上多加努力，就不会那么容易受伤害了。
花箴言 : 最美的爱情是不求回报的爱情。

12月31日 桧木 (Japanese Cypress)
花语 : 不灭
花占卜 : 您是个耐力强，又吃苦耐劳的人，有清晰的人生目标。您那义无反顾的踏实态度，对旁人起了激励性的作用，您对真理的追寻，吸引了许多有同感的同伴。您对爱情的执着，连本来无意的人也被您的热情融化。
花箴言 : 木材的好坏，在于使用者如何应用而已。
