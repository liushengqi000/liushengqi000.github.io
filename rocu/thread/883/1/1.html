在Acrobat 5.0中写字的方法：
方案一：
1、 先用Form Tool在要填写的地方画一个方框。随便起个name；
2、 选择Text Select Tool，点击方框处，就可以写字了！
注意如果你选择了Text Select Tool点击方框后出现一个对钩或者一个按钮，那么就是
你的Form Tool中的Type选择了check box或者combo box等，改为Text就行了
方案二：先选择Touchup Text Tool，然后用鼠标点击要写字的地方同时按住Ctrl键，这
样就可以写字了！  

在Acrobat 5.0中打钩的方法：
1、先用Form Tool在要划钩的地方划个方框
2、在弹出的对话框中在name中随便写一个名称
3、在name右边的Type中选上check box
4、去掉appereance中border color和background color前的对钩
5、在options中check type选check
6、在options中export value选yes
7、千万记住最后在export value下面的default is checked前划对钩
8、点击Ok并且选上除From Tool以外的图标后一切就OK了


使用说明：下面是大家填表时常遇到的打勾问题。
这里介绍的是acrobat5.0版本，对于其他版本，读者可以自己找到相应的工具，
我试过3.0以后版本，都有此工具。
Acrobat的基本概念是图层，也就是用户可以把上面的文字看作一个图层，如果
需要在上面添加文字或者做其他处理，那就再添加图层，在你所添加的图层上面进行操作
即可。
1、添加复选框或文本框
点击表格工具，在图1中所示是选择复选框的界面，复选框的属性也可以在此页
面设置，使用者自己尝试。
2、选中，显示对号
添加复选框后，将鼠标移开，并点击小手一下，再点击复选框，复选框将出现对
号。
3、在文本框内输入文字
添加文本框后，将鼠标移开，并点击小手一下，再点击文本框，文本框变成可输
入状态，使用者就可以输入文字了。
4、调节复选框大小、修改复选框属性
点击表格工具，复选框变成刚刚添加时的红色，这是对号消失（对于文本框，里
面的文字也消失）。鼠标右键点击，则可以看到1步时所见对话框，在里面就可以修改它
的属性了。

需要注意的事项有这么几点：

1、无论是作为复选框对象或者其他如文档编辑对象，都需要设定该对象的名字，这个名
字是不要重复的，当几个相同名字的复选框或文本编辑框存在时，他们同步修改。比如A1
和A2为同名的复选框，那如果开始时A1为选中状态A2为未选中状态，则再一次点击A1或者
A2时，他们将分别同时
改变为A1未选中，A2选中。当然有时候为了让许多同样状态的复选框同步操作，也可以设
置同名，同时要注意他们的开始选中状态。对于文档，则对需要出现相同内容的文档采用
同名方法，则你只需要填写一个文本框即可。
2、实际上我们添加的文档框和复选框都是再添加一个图层完成的，因此如果需
要准确定位，则在添加这个图层时需要适当调整它的大小。当然，你点击这个复选框后，
会在各个角以及边的中点出现红点，将鼠标移上去变成可调节状态即可拖拉调节大小。
3、复选框和文本框等有颜色的属性设置，请调节为背景颜色或透明颜色，如果
是白底黑字，默认即可。


首先下载Adobe Acrobat 4 Full Version
下载的地址有：
ftp://166.111.136.3/pub/TextTools/Editor/Acrobat/4.0
ftp://tracy.bupt.edu.cn//ComeInH ... Reader_4.0/ZipFiles
ftp://ftp.cs.uestc.edu.cn/pub/System/View/Acrobat
ftp://ftp.sepc.ac.cn/pc/text_tools
.........
总之有很多地方下载了，找个搜索引擎(166.111.136.3)搜搜就知道了。
记得还要下载一个Chinese Font Toolkit，一般和它在同一个目录下。
然后安装，打开需要编辑的PDF表格。

PDF格式的表格主要有两种编辑方法。

一：使用Touchup Text Tool
选择左边工具条中最下面一组第一个工具（一个空心的T)，然后在需要输入文
字的地方左击即可。如果没有反应，可以按住Ctrl再点击。如果要更改字体以及文
字大小，可以用鼠标选择填写的文字，右击，选择Attributes...，就可以了。

二：使用Text Annotation Tool
选择左边工具条第五组第二个工具，在申请表格上拉出任意大小的一个框，然
后就可以往里面填写文字了。如果要改变文字字体和大小，先把鼠标在文本输入框
之外的地方先点一下，再移回来，会看到鼠标变成一个箭头模样，右击，选择
Properties...就可以了。鼠标变成箭头模样的时候还可以将文本框拖动到任意地
方。

还有其它的方法，比如Form工具，利用PhotoShop等，我试了试，都比较麻烦。上
面两种方法应该可以解决所有的问题了吧。

偶发现用Touch up tool时，会把下划线冲掉，而且必须在字体符合的情况下才能成功。

偶经过尝试，发现用Form Tool填表快捷方便，不会有上述麻烦。。操作如下：
1.启动Acrobat完全版
2.打开文件
3.选用Form Tool（在快捷工具栏的倒数第三个），鼠标变为“十”字形，在需要填表的
下划线上方拉出一个合适大小的长方形框，接着跳出一个对话框，在name一栏随便输入后
确定。
4.选用Hand Tool（在快捷工具栏的第一个），然后在刚才的长方框处（边框已无，大概
估计一下）输入信息。
5.大功告成。
