SEPTEMBER


9月1日 老虎花 (Tiger Flower)
花语 : 爱我
花占卜 : 您是个细心而又喜欢照顾他人的人，对方感激您，您亦得到欣慰。您喜欢帮人扯红线，热心助人，但面对自己的问题却束手无策，您希望得到别人的呵护，却不懂得表达，有时也会意气用事，对爱情缺乏耐性。
花箴言 : 真爱一生才有一次，您应该睁开眼睛看清楚。


9月2日 墨西哥常春藤 (Mexican Ivy)
花语 : 事非
花占卜 : 您性格鲜明，言论突出，故此容易成为被人议论的对象。您非常介意的评价，更
会因为谣言而令自己无法自拔。如果您能收敛一下自己率直的性格，慎重行事，那么就可以
减少别人对您的误解。
花箴言 : 凡事适可而止，率性子也是一样。


9月3日 雏菊 (Marguerite)
花语 : 爱在心中
花占卜 : 您对自己缺乏信心，对爱情持怀疑的态度，经常占卜问卦，只为心安理得。您好
幻想，又相信自己的幻想，为一点小事也会和情人吵闹、分手，藉此测试对方是否爱您，这
种伎俩只可偶一为之，弄假成真时便后悔莫及。
花箴言 : 任性是不能解决爱情疑惑的。

9月4日 萝卜草 (Geum)
花语 : 满载希望
花占卜 : 您有才能，处世圆滑，遇到困难可以迎刃而解，前途可说一片光明。要使事业发展顺，必须贯彻始终，改掉半途而废的恶习，加上理性的坚持，这样才会得到别人的支持和信任。
花箴言 : 您需要送花给情人来传达您的心意。

9月5日 榆木 (Elm)
花语 : 信赖
花占卜 : 您是个耿直的人，重视彼此间的诚信，一旦被信任的朋友背叛时，那种伤心打击比一般人严重十倍，因此，您不会轻易相信别人。若果真是如此的话，您也有损失，不再信人，也就是令自己没有希望。
花箴言 : 有人说  :「女人就如藤树般缠着男人，两者不会轻易分离。」

9月6日 旱金莲 (Nasturtium)
花语 : 爱国心
花占卜 : 您是个独立冷静的人，思想成熟，从不倚赖他人。乍看之下，别人误以为您是个极难相处的人，但事实上您非常照顾家庭，只不过欠缺细心的关心，对待爱情比较被动和冷感。
花箴言 : 爱情无处不在，只是您心不在焉。

9月7日 橘花 (Orange)
花语 : 新娘的喜悦
花占卜 : 您是一个善良的人，心胸广阔，不易与人结怨。但您耳根软，容易被人说服，小心成为骗子的下一个目标。幸好您是受幸运之星眷顾的人，从恋爱到结婚，总有幸运之神帮您，无后顾之忧。
花箴言 : 不要过份放纵自己，不然会弄巧反拙的。

9月8日 芥子花 (Mustard)
花语 : 小幸福
花占卜 : 您是个有谋的人，尤其在投资方面，能以最少的本钱赢得最大的收获，而且眼光准确，有长远的策略。您最大的缺点就是给人一副吊儿郎当的印象，没有安全感，异性对您总是若即若离，是时候要修正一下自己。
花箴言 : 少女情怀总是诗，她经不起漫不经心的对待。

9月9日 蒲菊 (Michaelmas Daisy)
花语 : 追忆
花占卜 : 您是个念旧的人，对于遥远的往事，仍历历在目，久久未能忘怀。虽然如此，您对未来仍充满信心，感性加上知性的您，一定能遇到一位投缘的人，舒缓心?面积存已久的压力。
花箴言 : 您的心事对方早已看出来了。

9月10日 白色翠菊 (China Aster)
花语 : 诚信
花占卜 : 您平时不苟言笑，熟络之后才会发觉您其实是个活泼好动的人，外表的朴实，只是一个表面的您。您对人很忠心，期望纯洁的爱情，日后多点培养自己的内涵，必会遇到理想的情人。
花箴言 : 不要掩饰了，将自己的真性情流露出来吧。


9月11日 芦荟 (Aloe)
花语 : 合作
花占卜 : 您总是背负着一个很大的包袱，?面装满感情的困扰，令您精神受损，消沉而无生气。其实您是一个很坚强的人，勇于承受挑战和压力，只要您有坚定的意志，一定可以冲破心理障碍，重现昔日的光采。
花箴言 : 爱的伤痕正是爱的记号。

9月12日 铁线莲 (Clematis)
花语 : 心之美
花占卜 : 您有一颗纯良的心，从不懂欺骗别人，单纯地寻求一种如父母般爱护您的友情或爱情。您天生直觉敏锐，帮助您很快找到意中人，但您容易被甜言蜜语诱惑，走上危险的道路。
花箴言 : 当爱火在心中燃点时，是最光芒亦最具杀伤力的。

9月13日 杨柳 (Weeping Willow)
花语 : 纯朴
花占卜 : 您时常满脸笑容，对人活泼大方，容易获得他人好感，而且处事圆滑，使您的人际关系得到扩展。当您恋爱时，您会全情投入，但不懂得全身而退，遇到挫折更是意志消沉，您的欢乐情绪容易受他人影响。
花箴言 : 月圆之夜份外迷人，同时亦会带来伤感。

9月14日 温桲
花语 : 诱惑
花占卜 : 您拥有超凡的能力和光明磊落的人格，是天生的领导者。您对人不设防，因此诱惑亦特别多，而您随机变的能力，摆脱了不少无谓的缠绕。您将有不错的儿孙运。
花箴言 : 遇事切勿急进，三思而后行是保护自己的良策。

9月15日 天理花 (Dahlia)
花语 : 华贵
花占卜 : 天生的贵族分子，自尊心重，不会轻易吐露真情，但渴望被人爱护，为找寻被爱而努力。您过于自我的性格，常常怀疑对方是否真心爱您，为了验证而朝三暮四，再这样下去难免会寂寞一辈子。
花箴言 : 被宠爱的人是天之骄子，亦容易成为寂寞者。

9月16日 龙胆草 (Gentian)
花语 : 忧郁
花占卜 : 您有很好的人缘，当您面临苦困时，您身边的朋友会支时和帮助您，给您克服困难的力量。您很有正义感，鄙视那些背信弃义的朋友。您眉宇间的淡淡忧郁，正是您的魅力所在。
花箴言 : 即使不被认同，亦不可自轻自贬。

9月17日 石南花 (Heath)
花语 : 孤独
花占卜 : 您内心充满热情，为寻找爱情而积极行动。您外表看来似漫不经心，还带有点孤独的意味。别再装模作样了，恋爱是光明正大的事情，母须躲躲藏藏，愁眉苦脸的样子并不适合您。
花箴言 : 爱是落落大方的，母须闪闪缩缩。

9月18日 蓟花 (Thistle)
花语 : 严格
花占卜 : 您是个独立自主、自我保护意识很强的人，紧守严以律己、宽以待人的原则，因此受人尊敬。您的情人就在众多的仰慕当中，只要您将紧张的心情放轻微一点，就会发现对方的存在。
花箴言 : 有人懂得欣赏您，总比自己欣赏自己要好。

9月19日 蓑衣草 (Carex)
花语 : 自重
花占卜 : 您天生有点小聪明，拥有丰富的联想和创造力，非常受人欢迎，亦容易造成自满。由于您爱恨分明，不善掩饰自己的情绪，影响了您的人际关系，要保持和气，就要戒掉您那种骄傲的性格。
花箴言 : 人类最伟大的力量是懂得笑。

9月20日 迷迭香 (Rosemary)
花语 : 想起我
花占卜 : 您对人亲切，友善，善于关心，安抚别人的情绪，是个很好的倾诉对象。不过，您不擅长表达自己的情感，亦不懂得用撒娇、使性子等小手段来引起别人的注意，经常独自神伤，若找到真正了解您的人，您的生活将充满愉快。
花箴言 : 您只想编织一个美丽的爱情梦。

9月21日 秋水仙 (Autumn Crocus)
花语 : 青春无悔
花占卜 : 您拥有一段光辉灿烂的青春时光，带点野性及反叛的行为，不愿成为随波逐流的潮流跟风者。您天性多疑，凡事必先经过认真考虑，才会付诸行动，加上您对人冷傲，总有患得患失的心态，尽管如此，您对青春仍然无悔。
花箴言 : 花开堪折直须折，莫待无花空折枝。这道理您应明白。


9月22日 小判草 (Quaking Grass)
花语 : 兴奋
花占卜 : 您是一位好辩论、易激动的人，您最喜欢热闹，有时会得意忘形，乐极生悲，事后后悔不已。虽然您的理解能力很高，但处事冲动，久缺周详的计划，难有好结果，尤其是感情，爱是需要互谅互让，不是听您说道理的。
花箴言 : 激动时需要深呼吸，纾缓激动的情绪。

9月23日 紫杉 (Yew Tree)
花语 : 高尚
花占卜 : 您有优雅的气质，雍容的仪态，配合年岁的增长，散发出一股成熟的动人魅力。但您有一副冷脸孔，使人不敢贸然亲近您，就像冰山上的雪莲，可望而不可及。如果您不把自己的真心真意交出来，恐怕爱人也会离您而去。
花箴言 : 冰山美人是很难找到合适的对象的。

9月24日 橘子 (Orange)
花语 : 新娘的喜悦
花占卜 : 您有一颗纯真善良的心，愿意分担别人的忧愁，这样的您，总是给人一股清新的感觉，并且充满希望。在爱情和事业方面，会非常顺利，可以找到如意郎君。但在钱银方面您应小心，因为您没有理财概念。
花箴言 : 虽然您不在乎钱财的重要性，但不要被它累了您。

9月25日 黑麦 (Aninated Out)
花语 : 爱的音符
花占卜 : 您是个极感性的人，加上丰富的创意，您的生命有如一篇动人的乐章，经常如沐春风。您的谈吐、举止得体，朋友众多，是个受欢迎的人物。但凡事反应过敏会遭人讨厌，应学会宽大些，容纳更多的空间。
花箴言 : 生活自有其目的，不必刻意找寻目的。

9月26日 柿 (Date Plum)
花语 : 自然美
花占卜 : 您自少对大自然的奥妙产生强烈的兴趣，热爱大自然的壮美，因此虚怀若谷，心
胸广阔，对于所有的困境挫败，自有承担的魄力。加上您擅长交际，思想灵活，必定可与心爱的人建立美满幸福的生活。
花箴言 : 人生百态，不必拘泥于某一点上。

9月27日 橡树 (Oak)
花语 : 永恒
花占卜 : 您有一颗宽大的心，凡事看得开，天生的乐观精神，令您的人生充满欢乐。但您必须加强自己的内在潜质，接受新知、充实自己，这样才是永恒的欢愉，切勿捕鱼五天晒网，浪费光阴。
花箴言 : 万丈高楼平地起，根基稳固是最重要的。

9月28日 雁来红 (Love-Lies-Bleeding)
花语 : 爱戴
花占卜 : 您有独特的风格，外刚内柔，嘴巴经常嚷着要接受挑战，内心却并不那么好强，只是要面子罢了。事实上您是个深情诚实的人，但总是惹人误会，您要知道，办不到的事情就别逞强，以免招人话柄。
花箴言 : 朋友易得，知己难求。

9月29日 苹果花 (Apple)
花语 : 名声
花占卜 : 心地善良的您，为大家带来清新爽朗的气氛；由于缺乏自信，对人总是言听计从，您看来柔弱，实际上是外柔内刚，不会轻易妥协。您爱情存有幻想，希望被宠爱和追求，私底下其实早已有了主张。
花箴言 : 对待爱情要真心真意，虚情假意只会自讨苦吃。

9月30日 杉 (Cedar)
花语 : 雄伟
花占卜 : 您是个充满自信与斗志的人，自己认为对的路一定会坚持走下去，绝不会轻易妥协屈服。您周围的朋友好坏参半，有的伺机插您两刀，有的随时助您一臂之力，只要您谦虚点，不要过于自满，厄运是可以避过的。
花箴言 : 您应谨记「满招损、谦受益」的良言。
