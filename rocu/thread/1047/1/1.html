JANUARY


1月1日 雪花莲 (Snow Drop)
花语   : 纯白的爱
花占卜: 您个性高傲，喜欢完美无暇的事物，容易被人误解为倔强、顽固及不容易妥协，实际上您是个吃苦耐劳、思想成熟的人。但是，您偶尔也会比较挑剔，您的高标准不是人人都可以做到的，所以应保持冷静与克制。
花箴言 : 爱上这天出生的人，需像亲吻雪花莲般她。

1月2日 黄水仙 (Narcisus Jonquilla)
花语   : 神秘
花占卜: 您就如仙人般拥有迷人的风采，这种动人的魅力，总会让人不自觉地迷恋上您，深深被您感动。您经常要在两者之间做选择，无论您选择了哪一个，都应该相信自己，坚持到底，让不可能的爱情，开出美丽的花。
花箴言 : 美丽的水仙总是多情善变的。

1月3日 藏红花 (Spring Crocus)
花语  : 执着
花占卜: 您是个完美主义者，对于感情非常认真和执着，一心一意追求完美的爱情与人生。清纯而高傲的您，并不适合勾心斗角的环境，您喜欢宁静平和的生活，倾听内心的呼唤，感受完美的剎那。
花箴言 : 若果人生没有了爱，就像在漆黑的隧道前行，没方向，举步维艰。

1月4日 白色风信子 (Hyacinth)
花语   : 游戏人间
花占卜 : 您给人一种清新温柔的感觉，外表随和活泼，但私底下很有自己的原则，相当好胜；您不会轻易附和他人的意见，做事很有主见，精于人际关系。不要太过嬉戏，无意中的一句话会带给您致命的伤害。
花箴言 : 爱无处不在，只是您随手可以抓到多少。

1月5日 雪割草 (Hepatica)
花语   : 耐心
花占卜 : 雪割草是一种耐寒的植物，在这天出生的您，亦同时拥有相当的耐性，不惧困难挑战，耐心地逐一攻破。您在感情上显得保守和被动，即使遇上心仪的对象，仍会故作镇定，是异性眼中的理想情人。
花箴言 : 被爱是一种恩惠，必须全心全意，全情投入去接受。

1月6日 白色紫罗兰 (Violet)
花语   : 洁白无瑕的爱
花占卜 : 您是一个很有魅力的人，而且纯真可爱，您的出现就如星星般璀璨，令周围的人都被您的一举一动所吸引。您容易陷入危险的恋爱，处理感情应三思而后行，切勿感情用事，率性而为。
花箴言 : 您是太阳与月亮的结合体，追寻如星星般璀璨的浪漫爱情。

1月7日 白色郁金香 (Tulip)
花语   : 单恋
花占卜: 您是一个很有理想的人，往往将一件平凡的事情美化，寻求心中神圣的爱。但因为您的高傲及自以为是，不会轻易将心声吐露，形成自我精神苦恋或单恋，您或许认为这样的恋爱才是忧郁、理想的爱。
花箴言 :  应慎金对待自己的感情，单恋是苦果，相恋才是感情升华。

1月8日 紫色紫罗兰 (Violet)
花语   : 爱情
花占卜: 您是一个成熟及智能型的人，做事必定深思熟虑，计划周详才采取行动，所以大多数能够一击即 中，散发智者的魅力。您很容易答应朋友的要求，反而忽略了自己的需要，记着千万不要不要从事博彩活动。
花箴言 : 想爱就要去爱，不必事事计较，否则便无法恋爱了。

1月9日 黄色紫罗兰 (Violet)
花语   : 纯朴
花占卜: 您是一个多疑、敏感的人，经常考虑各方面的得失，但做事绝不马虎，而且胆大心细，勇于尝试，经常都有意外的收获，加上您天生具备的勇气与斗志，令您感情、事业、人生，都有很好的发展。
花箴言: 积极、进取是成功的不二法门。

1月10日 黄杨 (Box-Tree)
花语   : 不屈不挠
花占卜: 您个性豪爽，绝不会为小事情烦恼，对人对事不过于执着，是个典型的无欲无求、淡泊名利者。您有无比的意志，不容易受外间引诱，但有时也会因为消极和被动而错失大好良机。
花箴言: 浪漫需要幻想，恋需要执着。

1月11日 匀桧叶 (Arbor-Vitae)
花语   : 善良
花占卜: 您散发出一种善良的魅力，为人热情，重视朋友，因此您很受同性和异性的欢迎。不过，您应要留心异性的态度，学会拒绝一些无谓纠缠，避免别人误会您一脚踏几船。
花箴言 : 恋爱是生命的根源，热情是爱情的催化剂。

1月12日 庭荠 (Sweet Alyssum)
花语   : 优雅
花占卜: 您承受了家族中独有特征，天生具有一种高贵典雅的气质，您自然拥有的内在美，令您在人前显得自信和高雅。您应该好好珍惜这份特质，切勿造作夸张，以免影响您的魅力。
花箴言 : 只要处理得体，您必定拥有美满的幸福婚姻。

1月13日 水仙 (Narcissus)
花语   : 自我
花占卜: 您懂得将自己收藏起来，不会强于表现自己的个性，您喜欢和平的生活，不会随便与人结怨。但是，您为了逃避伤害，遇事总是畏首畏尾，这样只会惹来更大的麻烦，您应该提起勇气，笑面迎挑战。
花箴言 : 水仙虽美，但总是那么早便凋零，惹人怜惜。

1月14日 报春花 (Cyclamen)
花语   : 含羞
花占卜: 报春花在圣诞节期间盛开，虽是欢乐的应节花，但有离别的意味。在这天生日的您，对人热情且埋头苦干，却因为过度热情而虽常被人误会，而您的成就亦常遭人嫉妒。只要您保持笑脸迎人，矛盾就会化解。
花箴言: 花开花落，属于您的季节，一定要把握欢乐的时机。

1月15日 刺竹 (Thorn)
花语   : 严格
花占卜: 带刺的竹子，给人一种刚直、具保护性的感觉。您的个性刚正一阿，严肃认真，受人尊敬，在现代人眼中有点古板。您对待爱人的态度，太过苛刻严格，缺乏情趣，不妨改变一下您的态度，会轻勿为人接受。
花箴言 : 对情人体贴、宽容一些，您的爱情会如花开般灿烂。

1月16日 黄色风信子
花语   : 竞争
花占卜: 您有积极进取的性格，不断地自我鞭策、力求进步，您处事有计划、有谋略，沉着应战，是一个很好的领导者。但在爱情路上，却不能那么死板，争先不一定有好处，偶然的退让，伴侣会觉得您有人情味一些。
花箴言 : 您有进取的一面，也有感情的一面，淡紫色的花，总会令您心颤。


1月17日 酸模（Sorrel）
花语   : 挚爱
花占卜: 您是一个感情丰富的人，而且纯真善良，绝少会主动伤害别人。您喜欢恋爱，一旦坠入情网，您会浑然忘我，对于所爱的人，更会付出双倍的爱心，恋爱使您变得充实、满足和温柔。
花箴言 : 爱情一浪接着一浪，那么浪漫一生又何妨。

1月18日 印度锦葵 (Indian Mallow)
花语   : 猜测
花占卜: 您很在意别人对您的看法，时时怀疑别人是否正在谈论您，令您做事总是诸多顾忌，难有自己的独立主张。您应该放开怀抱，学习建立自信，体让别人的难处，亦保持自己的原则，那便可以减少您猜忌的毛病。
花箴言 : 爱是一种包容和恩惠，不可以用猜忌、怀疑来对待的。

1月19日 松 (Pine)
花语   : 健康、长寿
花占卜: 您很有自己主见，不现会世俗的眼光和束缚，崇尚自然悠闲的生活，对爱情很执着，愿意为爱情奉献。但您要注意，您的正义感和勇敢，可能会令您变得清高、骄傲而离群，而且是非不绝。
花箴言 : 高风亮节是一种美德，珍惜自己又怎会是罪恶？

1月20日 金凤花 (Butter Cup)
花语   : 童真
花占卜: 幸运之神总是眷顾着您，您既得到梦中情人，在财运上亦是事事如意。您有很大的野心，同时自尊心亦很强，也许因为这样，身边的人会以为您是高不可攀而疏远您，您可能会觉得孤独和离群。
花箴言 : 自尊与自强是进取的勤力，也是人际关系的阻力。

1月21日 常春藤 (Ivy)
花语   : 沉静、安详
花占卜: 深情且真挚的您，遇到好对象当然身受其惠。您对于浪漫总是比别人有多一倍的执着，您相信并寻求天长地久的爱情，一旦给您遇上了，就会死心塌地爱下去，所以在择偶时应该小心。
花箴言 : 爱是需要包容和忍受，人因为爱而有许多地方可以原谅。

1月22日 苔藓 (Moss)
花语   : 母爱
花占卜: 您喜欢顺其自然的生活方式，凡事随遇而安，对人关怀备至，带给人温暖和谐的感觉，和您相处的人，都感觉轻松愉快，不会有压力，您的情人更被您的温柔深深吸引，而您亦享受温馨恬适的爱情。
花箴言 : 爱是不惜一切的奉献，以及真心真意的占有。

1月23日 芦苇 (Bulrush)
花语   : 服从
花占卜: 您虽然拥有超凡的、先知自觉的能力，但不懂得控制自己的情绪，做事总是争先恐后，不会计划，给人一种轻浮、鲁莽的感觉。如果您相信自己的判断没有错的话，更应听取别人的意见，多方引证，对您百利而无一害。
花箴言 : 先知先觉是天聪，后知后觉是迟钝，不知不觉是麻木。

1月24日 番红花 (Saffron Crocus)
花语   : 进退得宜
花占卜: 您应该戒除奢侈浪费的习惯，紧守勤俭朴素的美德。您处于一个重要的地位，而且有使命感，相信总有出人头地的一天。您虽然受赏识，但也背负着不少压力，您应控制自己的脾气，消除压力是有很多方法的。
花箴言 : 凡事适可而止，遵守得饶人处且饶人的道理。

1月25日 鼠耳草
花语   : 纯真
花占卜: 您个性纯真，不识世途险恶，容易爱上不该爱的人。您有很好的条件质素，但不会运用，令情绪陷入困境，一厥不振。您应睁开眼睛，选择良师益友，那些活泼开朗的朋友，才是您的良伴。
花箴言 : 最不在乎爱情的浪子原来就是您自己。

1月26日 含羞草 (Humble Plant)
花语  : 敏感
花占卜: 您有令人无法抗拒的魅力，个性率直，反应敏锐，很多事情都逃不过观察。但是，您有时亦颇为软弱。总是唉声叹气，只要提起勇气，努力不懈，坚持自己的理想，问题是可以解决的。
花箴言 : 幸运就像水晶球般亮丽易碎，请把握、珍惜。

1月27日 七度灶 (Mountain Ash)
花语 : 坚固
花占卜 : 据说此树之坚硬程度，即使放于七座炉灶中也烧不断。于此日出生的人相当自信，对人要求颇为严格，在力求进步中缺少了人情味。您应该放松一些，不要刻意掩自己，偶尔间的过错并不是什么大不了的事情。
花箴言 : 不要因为偶然间的说话失误，而放弃了应有的幽默风趣。


1月28日 黑色白杨木 (Black Poplar)
花语 : 勇气
花占卜 : 您是个非常有勇气的人，坚持自己的理想，不易受人影响。在旁人眼中，您是个固执己见的人，专横独行，对您总是七分尊敬，三分远离。您应学会尊重别人，集思广益，将您的长处发挥出来。
花箴言 : 不要老是想做众人的主角，事实上您就是自己一生中的主角。


1月29日 苔藓 (Moss)
花语 : 呵护
花占卜 : 苔藓覆盖着岩石，就像母亲呵护着孩子一样。在此日诞生的您，给人一种慈爱友善感觉，您对人总是一贯的温婉口吻，情人在您的拥抱中完全感受到被呵护和溺爱。您做事总是慢条斯理，绝不会逞一时之快。
花箴言 : 自信是成功的首要条件，相信自己才可以冲破困难。

1月30日 金盏花 (Marsh Marigold)
花语 : 盼望的幸福
花占卜 : 您就像所有的怀春少女一样，等待着白马王子的到来，期盼着幸福降临在您的身上。您的思想活泼开放，可以容纳不同的新鲜事物，您要知道幸福不是必然的，您需要主动寻找，幸福就在您的身边。
花箴言 : 努力争取，幸福就在不远处。

1月31日 黄色藏红花 (Spring Crocus)
花语 : 青春喜悦
花占卜 : 您就像春天的鸟儿，浑身充满青春的活力，个性活泼开朗，是典型的乐天一派。对于不如意事，您会很看得开，绝对不会愁眉苦脸，身边的人都被您的乐观精神感染，您的爱情将充满愉快欢乐的气氛。
花箴言 : 爱情像春天的花儿般盛开，美好得令人不顾一切。
