第一课   登录


本论坛可以在两个地方输入ID和密码登录，一个是主页上近下方的登录框，另一个是点击主页上方的“登录”。

这两种登录方式有所区别。

区别在于cookie保存的时间：

一．选择主页下方快捷的登录框

ROCU019主页下方登录.jpg (24 KB, 下载次数: 0)

下载附件

系统会默认将你的ID和密码用cookie储存在该电脑一年时间，除非按了退出，否则一年内打开若酷天下论坛就会自动登录。

二．选择点击首页上方的“登录”

ROCU020-按上方登录.jpg (11 KB, 下载次数: 0)

下载附件

ROCU021-登录即时.jpg (100 KB, 下载次数: 0)

下载附件

系统默认的是一年。如果用的是自己的电脑，一般都不用再选，就用这个默认的；如果用的是公共电脑，建议选择“即时”，每次走的时候不要忘记按退出，关闭电脑或者重启，以免你的帐号被他人盗用。

如果发现无法登录，可能是由于:

A.你的防火墙的问题，安全设置过高。

由于 Norton Internet Security 过于强大和苛刻的过滤,使用该防火墙软件产品的用户在其默认配置下访问 Discuz! 或其他网站将会产生一些问题.具体表现在:

1. 由于硬性屏蔽了来路信息,导致 Discuz! 提交安全检查无法通过,在发帖等情况下提示为 未定义操作
2. 由于误判断,将发帖时的 Discuz! 代码辅助提示判断为广告信息而不被执行

尽管如此, Norton Internet Security 仍不失为一款强大有效的网络安全产品,如果您访问 Discuz! 存在上述两种问题,我们建议您修改 Norton 的默认配置以避免可能给您带来的不便.方法如下(以 Norton Internet Security 2003 中文版为例):

1. 选项 =&gt; Norton Internet Security =&gt; Web内容 =&gt; 浏览站点的信息 =&gt; 选择 &quot;允许&quot;(默认是&quot;禁止&quot;)

2. 禁止广告 =&gt; 配置(在窗口右下方) =&gt; 取消&quot;启用禁止广告&quot;的复选框

B. 你的浏览器设置拒绝 cookies，点击浏览器上方“工具”，在“Internet选项”中，看第二个“安全”(security)和第三个“隐私”(privacy)，将这两个全部选择“默认”(default level)；

C.你的电脑的cookie设置问题。可以尝试换一台电脑看能否登陆，如果用的是windows xp系统，用“恢复”功能(system restore)将电脑恢复到一个能登陆的日子(注：恢复的是注册表，所以恢复后部分软件可能需要重装)

威震三山    500

第四课  头像


头像的选择有三种方式：

在控制面板“选择您的头像”一栏中

A：选定论坛提供的头像。

B：自定义头像：这个可以从电脑硬盘中直接上传。

C：指定网络中已有的地址来链接：前提是网上已经有了这个图片，将这个图片的URL（右键单击图片看属性中的地址）复制，然后粘贴到“完整URL头像位置”输入框里。

头像的大小可以在控制面板里面设置，只要不超过规定大小即可。

第六课  签名


签名档的设定会受积分的限制,积分越高签名档的长度越大。如果积分还不够，建议把不必要的空格和修饰颜色去掉，这样保证图片和说明文字的正常显示。

签名中的图片不能直接从你的电脑硬盘中上传，这张图片必须已经在网上有了才行。

如果你想把你硬盘里的一张图片作为签名，那么要先把它上传到网上的空间才可以。

你想把在网上看到一张图片用作签名，右键点击这张图片，看“属性”中的地址，那是个url，比如http://www.tom.com/pic ，把这个url复制到在控制面板的“个性化签名”输入框，前后加上img代码，即【img】http://www.tom.com/pic【/img】就可以了。

第八课  发表贴子


1.        发表一个新主题，获得20牛。
2.        发表一个新回复，获得10牛。
3.        贴子被加为精华，获得内力100牛和人品增加1。
4.        贴子被解除精华，减去内力100牛，人品降1。
5.        贴子被删除，内力与人品无影响。

加密与解除加密

在编辑贴子时，需要加密内容就点选文本输入区下面的相应选项，包括隐藏，加密，出售贴子。而要清除加密，编辑时，原来选了的地方却是没有打勾的，这时要在输入区里面手动删去正文内容最前和最后的加密代码。

设置威望代码：hide
设置回复可见代码：post
设置出售代码：sell

注意：只有主题贴能出售，回复贴不能。
